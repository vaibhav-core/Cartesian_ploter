# Cartesian_ploter

##Overview

Simulation of 2DoF cartesian plotter made with Gazebo and ROS2. 

All meshes are made in fusion with custum design.



##Launching

Download the whole repository as it is

In plotter.sdf change the file address of meshes in lines from to the address at which your files are saved make sure address are correct 

Navigate to the Codes folder of repository 

Launch using the command gz sim plotter.sdf
